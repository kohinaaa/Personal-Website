PROJECTS / WORKS

Here are some of the academic and personal projects I’ve developed — demonstrating experience in programming, web development, and system design.

Gantt Chart System

The Gantt Chart System is a desktop-based project scheduling tool written in Python. It uses Tkinter for the GUI and Matplotlib for chart visualization. This project demonstrates my ability to create interactive applications for time management and task visualization.

Jack En Poy Game

The Jack En Poy Game is a console-based game developed in C language. It implements conditional statements, random number generation, and loop structures to simulate a Rock-Paper-Scissors battle between the player and the computer. This project highlights my understanding of basic algorithms and logic flow.

View on GitHub

Pokédex System

The Pokédex System is a web-based application that retrieves Pokémon data using the PokéAPI. Built with HTML, CSS, and JavaScript, it allows users to search for Pokémon and display details such as name, image, type, and abilities. This project demonstrates my ability to work with APIs and asynchronous JavaScript.
